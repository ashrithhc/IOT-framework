Sample Works with Nest Integration

This application demonstrates best practices when integrating a Works with Nest product, such as:

  • Hiding the client ID and client secret
  • Providing a structure picker
  • Listening for changes
  • Detecting Home/Away
  • Including confirmation and error messages
  • Removing cached data and camera images
  • Handling deauth and revoked auth
